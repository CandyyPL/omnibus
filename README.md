# **Mechan Omnibus**

Quiz application written in React and Typescript made of 3 together-working instances:

- **Quiz app**: The main quiz platform for users with integrated questions from [DatoCMS](https://www.datocms.com/), and user information from [MongoDB](https://www.mongodb.com/)

- **Admin app**: A easy-to-use admin panel with allows admins to interview registrations requests and create new questions for the quiz app.

- **Server**: A [Node](https://nodejs.org/) server with makes connection with [MongoDB](https://www.mongodb.com/) available for both admin and quiz apps. It also handles the ranking, saving user information and many other stuff.
